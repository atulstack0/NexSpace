# Character model

Drop the converted character here as **`eric.glb`** (this exact path: `apps/web/models/eric.glb`).

The 3D client loads `models/eric.glb` on start. If it's missing or fails to load, avatars
fall back to the built-in styled "businessman" avatar — nothing breaks.

See **`../../CHARACTER.md`** (project root) for the 2-minute FBX → GLB conversion recipe and tips
on keeping the file small enough to commit/serve.
